Ferramentas Utilizadas

(Voltar)

Para preparar o ambiente para desenvolver sites, utilizaremos a ferramenta Visual Studio Code. Ela permite a instalação de plug-in ou extensões, e também utilizaremos o Live Server. Ele ajuda, por exemplo, a autocompletar padrões.

O navegador que utilizaremos será o Chrome, que possui algumas ferramentas de desenvolvimento exclusivas.

Inspetor de Elementos

(Voltar)

É uma ferramenta oferecida pelo Chrome, que apresenta recursos que nos permitem verificar como a página foi criada e como os elementos foram formatados.

Estrutura Básica do HTML

(Voltar)

Primeiramente, devemos criar uma pasta no Windows, que utilizaremos no VSC. A primeira página tem o nome "index" por padrão. As primeiras tags para de fato criarmos uma página são <html>, <head>, <title>, que é o título da aba do navegador e <body>.

Falando sobre Tags

(Voltar)

A tag <i> deixa o texto itálico e a <strong> por sua vez deixa negrito.

As tags <input> e <img> não são fechadas por outra equivalente.

Atributos das Tags

(Voltar)

Atributos são propriedades genéricas ou específicas. Entre os atributos genéricos estão o "id", que é um identificador de um único elemento, o "style", que é utilizado para colocarmos comandos CSS diretamente no elemento, e o atributo "class" padroniza um conjunto de elementos no CSS.

Entre os atributos específicos, estão "texto" ou "número" para a tag <input>, que cria um campo para o usuário digitar. O atributo da tag <img> é "src", onde colocamos o endereço de uma imagem da nossa pasta ou da internet. Já o "width" altera o tamanho dessa imagem.

Textos

(Voltar)

As tags básicas para textos são <h> para os cabeçalhos ou títulos em destaque e cujo tamanho varia conforme a numeração. A tag <h1> é a que formata o maior título, seguida por <h2> e assim por diante.

As tags <p> fazem a separação dos blocos de parágrafos. Podemos usar também <u> para sublinhar, <mark> para grifar em amarelo e <sup> para * ou marca registrada R, etc.

A tag <sub>, pelo contrário, faz uma marcação pequena, a <del> risca um texto e <abbr> muda a formatação da abreviação.

A <blockquote> é usada para formatar citações, como podemos ver no exemplo abaixo:

"Qualquer tecnologia suficientemente avançada é equivalente à mágica" - Arthur C. Clarke

Listas Ordenadas e Não Ordenadas

(Voltar)

A tag <ol> é utilizada para listas ordenadas e enumeradas, enquanto a <ul> (unordered) para listas não enumeradas.

Links

(Voltar)

A tag <a> é uma âncora para criar um link para outra página do mesmo site ou uma página externa. O atributo "href" indica a referência, ou seja, o nome da página ou o endereço do site.

O atributo target "_blank" abre outro navegador, enquanto o "_self" substitui a aba por outra. O atributo "title" indica um texto que nós escrevemos e que aparecerá quando a seta pairar o link.
